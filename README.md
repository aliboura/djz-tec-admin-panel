# Djz Tec admin panel

Djz Tec  is an  an admin panel based on bootstrap 4 and angular 9

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
git clone  https://github.com/aliboura/admin-panel.git
npm i
ng serve

```
