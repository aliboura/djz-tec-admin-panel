# Djz Tec admin panel

Djz Tec  is an  an admin panel based on bootstrap 4 and angular 9

## Installation

to install the Djz Tec admin panel :

```bash
git clone  https://github.com/aliboura/admin-panel.git
npm i
ng serve

```
